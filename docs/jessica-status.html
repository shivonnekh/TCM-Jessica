🌿 中醫師視角 — 建議加入咩功能

🔴 高優先 — 直接影響收入

購後追蹤 — 賣完之後主動問返

用戶買咗湯水，3–5 日後 Jessica 自動問：「之前個湯飲完點㗎？係咪有舒服啲？」
呢個係最直接嘅複購觸發。中醫調理需要持續，一次買完就消失係最大浪費。追蹤回饋 → 更新 CRM pain_points → 推薦下一款湯。
需要：CRM status = bought + timer → 主動廣播邏輯（天氣廣播基礎已搭好）。

🔴 高優先 主動推送 利用現有廣播架構

24 節氣養生提醒 — 最正統嘅 TCM 主動關心

中醫最核心係「天人合一」。每逢節氣（立春、夏至、秋分、冬至等），自動發一條 Jessica 嘅節氣養生 tip。
例子： 立冬 → 「今日立冬，係補腎最好時機，宜食黑芝麻、核桃、羊肉。Care Plus 有補腎系列，需要可以搵我 🌿」
比天氣廣播更有 TCM 特色，一年 24 個節氣 = 24 次主動觸點，完全零成本觸達。

🔴 高優先 主動推送 節氣 Calendar + 廣播

呢個係最自然嘅回流觸發，唔係賣嘢，係真心關心，而且會帶出新嘅購買決策。
